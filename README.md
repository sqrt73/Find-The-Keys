# Find-The-Keys
A CTF and Find The Object inspired game about the Linux terminal.

To play this game, it is recommended to use a Linux operating system for the best play.
If you are on a Linux distribution, this game is also best played in it's command line.

If you meet both of those requirements, then great!
Go to your terminal, type in `gh repo clone sqrt73/Find-The-Keys`, `cd Find-The-Keys`, and `./check.sh` to begin playing.
From there, it is up to you to find the keys using any method possible.
Good luck!
